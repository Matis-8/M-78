# M-78 Desktop Dictation App v1.0.0

The first official stable release of **M-78**, the premium AI dictation assistant for Windows.

## 🚀 Key Features
- **Offline-First Processing**: Discrete, secure, and fast.
- **Global Dictation**: Press `Ctrl + Alt + D` to dictate into any Windows application.
- **Waveform Widget**: A minimalist, high-fidelity waveform visualizer.
- **Premium Dashboard**: Track your dictation patterns and manage your workspace.
- **Smart Snippets**: Quick-inject reusable text blocks.
- **Custom Dictionary**: Auto-replace complex words and acronyms.
- **Privacy-First**: All data stays on your local machine.

## 🛠️ Improvements & Fixes
- Modularized architecture for extreme stability.
- Native Windows identity (Taskbar grouping, Premium icons).
- Silent startup support via VBS launcher.
- Robust focus restoration (prevents widget from stealing focus).
- Functional "Clear History" and "Reset Analytics" settings.

## 📦 Installation
1. Download **`M-78-Setup.exe`** from the assets below.
2. Run the installer.
3. Launch M-78 from your Desktop or Start Menu.
